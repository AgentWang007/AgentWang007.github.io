SQL Cheat Sheet

A Cheat Sheet for Commands

SQL is Structured Query Language which is a language used to operate a relational database. For this cheatsheet, we will be focusing on frequently used statements for operation. Please keep in mind that this is not complete.

Basic Query

SELECT column, ...

FROM table_name;

Conditional Query

SELECT column, ...

FROM table_name

WHERE condition;

Conditional Query

SELECT column, ...

FROM table_name

WHERE condition;

Pattern Matching

SELECT column, ...

FROM table_name

WHERE condition LIKE 'pattern';

Sorted Search

SELECT column, ...

FROM table_name

WHERE condition

ORDER by column_name

Sorted Search with Ascending or Descending

SELECT column, ...

FROM table_name

WHERE condition

ORDER by column_name DESC or ASC

Sorted Search with Between

SELECT column, ...

FROM table_name

WHERE condition

BETWEEN by integer and integer
